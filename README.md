# MapWithAR
## INT20H Mobile Test Task

### Как запустить:

`1. Открыть проект в Android Studio`
> не забудьте включить usb-debugging на телефоне

`2. Включить интернет и геолокацию на телефоне. Нажать "Run" в Android Studio. Когда приложение откроется, разрешить доступ к геолокации и камере`

> Если на карте не появилось ваше местоположение, перезапустите приложение через Android Studio

`3. Нажмите на карте на точку в какую вы хотите прийти`

> после нажатия на карте появится красный маркер

`4. Наведите камеру на пол и увидите белые точки (см. фото ниже), после этого нажмите на них`

![image](https://developers.google.com/codelabs/maps-platform/location-places-arcore-android/img/2a9b6ea7dcb2e249.png)

`5. После этого поворачивая камеру вы увидите ваше назначение, а точнее маркер в виртуальной реальности`
